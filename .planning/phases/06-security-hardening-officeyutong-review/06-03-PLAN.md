Harden receipt creation validation and make refund truly permissionless (Issue 2).

Purpose: Receipt creation doesn't cross-check pledge amount or type script hash — a backer could create a receipt with an inflated amount. Additionally, permissionlessRefund requires the receipt as an input, but receipts are locked with backer's secp256k1, making refund backer-only. Both are HIGH-severity mainnet blockers.

Output: Updated receipt/main.rs with pledge cross-check in creation, and updated builder.ts with receipt-free refund path.
</objective>

Rewrite `validate_receipt_creation()` to cross-check against the sibling pledge cell:

1. **Add pledge type hash to receipt type script args:**
   The receipt type script needs to know the pledge type script hash to identify the correct pledge cell. Store it in the receipt type script args (first 32 bytes). This means receipt type args = pledge_type_script_hash (32 bytes) instead of empty.

   Add args parsing at the top of `validate_receipt_creation`:
   ```rust
   // Load receipt type script to get pledge_type_hash from args
   let script = load_script().map_err(|_| ERROR_LOAD_SCRIPT)?;
   let args = script.args().raw_data();
   if args.len() < 32 {
       debug!("Receipt args too short — need pledge_type_hash");
       return ERROR_INVALID_ARGS;
   }
   let mut pledge_type_hash = [0u8; 32];
   pledge_type_hash.copy_from_slice(&args[0..32]);
   ```

2. **Find sibling pledge cell by type script hash:**
   Replace the current lock-args-length-based search with a type-hash-based search:
   ```rust
   // Find the pledge cell in outputs by type script hash
   let mut found_matching_pledge = false;
   for i in 0.. {
       match load_cell_type_hash(i, Source::Output) {
           Ok(Some(hash)) => {
               if hash == pledge_type_hash {
                   // Found a pledge cell — read its data
                   let pledge_data = match load_cell_data(i, Source::Output) {
                       Ok(d) => d,
                       Err(_) => return ERROR_LOAD_DATA,
                   };
                   if pledge_data.len() < 72 {
                       continue; // Not a valid pledge cell
                   }
                   // Parse pledge data
                   let pledge_amount = u64::from_le_bytes(
                       pledge_data[64..72].try_into().unwrap()
                   );
                   let mut pledge_backer_hash = [0u8; 32];
                   pledge_backer_hash.copy_from_slice(&pledge_data[32..64]);
                   
                   // Cross-check: receipt amount must match pledge amount
                   if pledge_amount != receipt.pledge_amount {
                       debug!("Receipt amount {} != pledge amount {}", 
                              receipt.pledge_amount, pledge_amount);
                       return ERROR_AMOUNT_MISMATCH;
                   }
                   // Cross-check: receipt backer must match pledge backer
                   if pledge_backer_hash != receipt.backer_lock_hash {
                       debug!("Receipt backer hash != pledge backer hash");
                       return ERROR_BACKER_MISMATCH;
                   }
                   
                   found_matching_pledge = true;
                   break;
               }
           }
           Ok(None) => continue,
           Err(SysError::IndexOutOfBound) => break,
           Err(_) => return ERROR_LOAD_DATA,
       }
   }
   ```

3. **Add new error constants:**
   ```rust
   const ERROR_LOAD_SCRIPT: i8 = 19;
   const ERROR_INVALID_ARGS: i8 = 20;
   const ERROR_AMOUNT_MISMATCH: i8 = 21;
   const ERROR_BACKER_MISMATCH: i8 = 22;
   ```

4. **Add necessary import:**
   ```rust
   use ckb_std::high_level::{load_cell_type_hash, load_script};
   ```
